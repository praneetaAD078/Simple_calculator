A basic web-based calculator built using Python and Gradio. This application allows users to perform simple arithmetic operations through an interactive web interface.

🚀 Features

Addition

Subtraction

Multiplication

Division

Handles invalid inputs and division by zero

🛠 Technologies Used

Python

Gradio

▶️ How to Run

Install the required library:

pip install gradio

Run the Python script or notebook.

A local web interface will open in your browser where you can enter numbers and choose an operation.

📂 Project File

calculator.py or calculator.ipynb – Main program file
